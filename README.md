# 🚀 Playwright Automation Project

## 👋 Sobre mí

Soy **Cristian Oliveros**, QA Engineer y desarrollador de software en constante evolución.
Estoy enfocado en fortalecer mis habilidades en **automatización de pruebas** para generar impacto real en productos y equipos de desarrollo.

---

## 🎯 Objetivo del proyecto

Este repositorio marca el inicio de mi camino con **Playwright**, donde busco:

* Aprender y dominar herramientas modernas de automatización
* Diseñar pruebas eficientes y escalables
* Reducir la cantidad de errores en aplicaciones
* Mejorar la calidad y confiabilidad del software

---

## 🧠 Contexto

Aunque ya tengo experiencia trabajando con **Cypress**, decidí ampliar mi stack y explorar Playwright por su potencia, velocidad y soporte multi-navegador.

---

## 🛠️ Tecnologías utilizadas

* Playwright
* JavaScript / TypeScript
* Node.js

---

## 🧪 Alcance del proyecto

En este repositorio estaré trabajando en:

* Automatización de flujos de login
* Pruebas end-to-end (E2E)
* Validación de UI
* Manejo de múltiples navegadores
* Buenas prácticas de testing

---

## 🚀 Cómo ejecutar el proyecto

```bash
# Instalar dependencias
npm install

# Ejecutar pruebas
npx playwright test

# Ejecutar pruebas con interfaz
npx playwright test --ui
```
